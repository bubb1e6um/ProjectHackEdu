# HackEdu 🖥️

**HackEdu** — это интерактивная обучающая среда с элементами stealth-головоломки, созданная для визуализации работы классических структур данных (деревья, графы) и алгоритмов. 

Игрок управляет объектом в 3D-пространстве и использует внутриигровой терминал (эмулятор UNIX-подобной консоли) для взаимодействия с окружением: работы с виртуальной файловой системой, сканирования локальных сетей и удаленного взлома патрульных дронов.

## 🛠 Технологический стек
* **Движок:** Unity 3D
* **Язык программирования:** C#
* **Архитектура:** MVC, ООП (паттерны Composite, Command, Singleton, State)

## ✨ Ключевые особенности
* **Виртуальная файловая система (VFS):** Кастомная реализация файловой структуры в оперативной памяти на базе деревьев (паттерн Компоновщик). Поддерживает создание директорий, чтение файлов и рекурсивный вывод структуры (`tree`).
* **Сетевая топология (Графы):** Механика локального сканирования вычисляет евклидово расстояние по 2D-плоскости для обнаружения ближайших узлов (серверов, дверей, дронов) и установки SSH-соединения.
* **ИИ противников:** Поведение патрульных дронов строго регламентировано Конечным автоматом (FSM), что изолирует логику патрулирования от состояния тревоги.
* **Бесшовный игровой цикл:** Интегрированный `GameManager` (Singleton) обеспечивает переход между уровнями без потери контекста.

## 💻 Доступные команды терминала
Внутриигровой терминал поддерживает базовые команды для работы с системой и сетью:
* `help` — вывод списка доступных команд.
* `ls` / `cd` / `cat` — навигация и чтение файлов в VFS.
* `mkdir` / `tree` — создание папок и рекурсивный просмотр файлового дерева.
* `scan` — поиск уязвимых устройств в локальном радиусе.
* `ssh <IP>` — удаленное подключение к узлу сети.
* `unlock` — обход блокировки подключенного устройства.
* `next` — переход в следующий сектор (уровень) при успешном взломе точки эвакуации.
* `clear` / `restart` / `quit` — системные утилиты.
